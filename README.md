# ThIntA

- - -

## Overview

ThIntA (Theoretical Integer Algebra) is a performant library for number theory, prime numbers, curve arithmetic, etc.
Currently, this is purely a learning project, so it is not recommended for use. It also requires a C23 compiler and very
recent CMake.
